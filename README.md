Dark.Raids
==========

Features:
---

* World Markers indicator
* Interrupt Announce
* Threatmeter
* Cooldown announce


Slash Commands:
---

* /dark [int|interrupt] channel party
* /dark [int|interrupt] suffix 'Memmoria next.'
* /dark [int|interrupt] whisper <player_name>
* /dark [int|interrupt] enable
* /dark [int|interrupt] disable
